# SRP-DIP-OCP

## 실행방법

1. 의존성 패키지 설치

```bash
npm install
```

2. env 파일 세팅

- REACT_APP_BASE_URL environment 설정필요
- .env.sample 파일 참고

3. development server start

```bash
npm start
```
